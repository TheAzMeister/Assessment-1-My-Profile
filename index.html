Project Idea

I want to create a program that searches the internet for information about any game ever made and put it on one place. All the information could be viewable via a website or app. The database will hold basic information about games like the cast, who made it, trailers as well as more technical stuff like a list of every update the game has had with a list of patch notes. It will even have information about what development was like. It will automatically search the web to find interviews and articles about the game and its developers. The aim is to have as much information about a game in one place.

In the future it is entirely possible that we may never be able to play the games we love in a legal way. This may be due to companies removing the servers from games or removing internet access to consoles so we can’t update games, or like what Ubisoft are doing, making it so you can’t download games and DLC ever again. If you don’t own a physical copy of a game, then there is a chance that it may be taken off steam or origin and you may never be able to play it again as legally we don’t own these games, we rent them from these companies so they can do whatever they want with them. I don’t want any games to be lost to history.

The user will search up any game they want, and they will be greeted with a page of information about the game. The program will search the internet looking for certain material about the game and display it on a website/app. Basic stuff like developers, cast, credits, publisher, suggestions, and reviews will show up. But more important and interesting thigs will show up like a list of every patch along with gameplay for each patch, list of known bugs, interviews on certain topics from developers, places you can buy it, deals for it, details about whether the game is available anymore, and many more. The program will try to gather as much information as possible and display it in one place, this means the user can ask to the program to collect certain data for them and then download it straight to their computer (this info may be too complicated/too much to be displayed on the website/app). It will also save all the information so that there is always an archive of the information so that even if we can no longer play a game we will still have as much data about it as possible so that we never ever forget that it existed. The program will also collect data on companies and publishers and rate them based on how bad they are. If they have good business practices, they will have a higher rating. The program will also search for how good the development of the game was and it will display that information with a simple graphic somewhere where the end user can’t miss it. For example, it might show a thumbs up or thumbs down near the name of the game to let users know if it was a good or bad development, it could show something silly like a gif of teeth crunching down on a disk to show if the developers were under crunch. If the game is no longer available anywhere to play, then user will be able to download the game off the website/app (with the company’s permission so that it is legal). They could also download alpha builds and beta builds as well a build for each patch. This is so we can see how the game is made and to try and see the thought process of the developers. It is also so we can keep track on which games are released finished as this is something important that should be documented as it can show how far a game has come and how much work the developers have put into the game, or it can show the greediness of some companies. This tool will document every aspect of games, the good and the bad.

We will use Atom as a place to code on as well as we can code together on it and it have lots of packages so we can customise it to how we like. We would use flutter to create the app and the website as its used by big companies such as google, eBay and BMW and it can be used on multiple devices. It is also open source. We will need a cloud database like mongo to store all the information as it would be too expensive to do it ourselves and we don’t have the space.

We will need skilled programmers who will design and create the program as well as a trusted company who is secure and can store all the data from the program. We will also need website/app developers and designers, so the app runs smoothly and looks great. It might be a bit hard to find programmers and designers as the project may be quite difficult for first year students however a more basic build would be much easier possible to find some programmers. We can easily find a server hosting website that is secure with a quick google search.

If the project is successful, then there will be a program that will search the internet looking for everything there is to know about a certain game so that even if a company tries to remove that game it will never be truly gone. By carefully documenting the development process it would be possible to shame companies into adopting better practices as their actions will be immortalised in our program for the entire world to see.
